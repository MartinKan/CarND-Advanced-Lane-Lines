Background
---
In this project, my goal is to write a software pipeline to identify the lane lines in a video.

Overview of Repository
---
This repository contains the following files that I will be submitting for the advanced lane finding project:

1.  [detect.py](https://github.com/MartinKan/CarND-Advanced-Lane-Lines/blob/master/detect.py)
2.  [project_writeup.md](https://github.com/MartinKan/CarND-Advanced-Lane-Lines/blob/master/project_writeup.md)
3.  [final_video_out.mp4](https://github.com/MartinKan/CarND-Advanced-Lane-Lines/blob/master/output_images/final_video_out.mp4)

The first file is the lane line detection program that I have written for the project.

The second file is a written report for my project that addresses each of the rubric points for the project.

The third file is the final video output when the project_video.mp4 file was fed through my lane line detection program (detect.py).
